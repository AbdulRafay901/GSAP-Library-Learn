# 🚀 GSAP Practice Project

A collection of animation experiments built while learning the GSAP (GreenSock Animation Platform) library. This project covers fundamental animation concepts, timeline sequencing, custom cursor interactions, and scroll-based animations using ScrollTrigger.

## ✨ Features

- Basic GSAP Animations
- From & To Animations
- Stagger Animations
- Timeline Animations
- Navbar Entrance Animation
- Custom Cursor Effect
- Hover Interactions
- ScrollTrigger Animations
- Pinned Sections
- Scrub-Based Scrolling Effects
- Responsive Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP
- GSAP ScrollTrigger

## 📚 Concepts Practiced

### GSAP Basics

- `gsap.to()`
- `gsap.from()`
- Duration & Delay
- Rotation & Scaling
- Repeat & Yoyo Effects

### Timeline

- Animation Sequencing
- Navbar Reveal Animations
- Staggered Elements

### Custom Cursor

- Mouse Tracking
- Hover States
- Dynamic Cursor Effects

### ScrollTrigger

- Trigger-Based Animations
- Scrubbing
- Pinning Sections
- Scroll-Driven Text Animation

## 🎯 Purpose

The purpose of this project is to practice GSAP and understand how modern web animations are built using timelines, interactive effects, and scroll-based triggers.

Created for learning and experimentation with GSAP animations.
